# nvim-config
Configuration for nvim

## Prerequisites
- [neovim](https://github.com/neovim/neovim)
- [ripgrep](https://github.com/BurntSushi/ripgrep)


## Install
```
./init.sh
```

## Package Manager
- [lazy.nvim](https://github.com/folke/lazy.nvim)

## Themes
- [catppuccin](https://github.com/catppuccin/nvim) - Default
